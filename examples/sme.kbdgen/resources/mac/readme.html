Keyboards for Northern Sami

This installer contains Northern Sami keyboards from the Divvun group at the UiT Arctic university of Norway. Further information can be found on the home page of the Northern Sami keyboad, including a picture of the keyboard layout.

Included keyboards

The installer contains X keyboard layouts:

  • for Y, based on the standard Y keyboard

This means that everything that is not specific to Northern Sami is exactly where it is on the standard keyboard.
